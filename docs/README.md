# SuperDimmer Documentation

This folder contains all documentation for the SuperDimmer Mac app project.

## Folder Structure

```
docs/
├── product/               # Product requirements and specifications
│   └── PRODUCT_REQUIREMENTS_DOCUMENT.md
├── reference-apps/        # Analysis of competitor/reference applications  
│   ├── REFERENCE_APPS_ANALYSIS.md
│   └── REFERENCE_APPS_DEEP_ANALYSIS.md
└── research/              # Market and technical research
    └── MARKET_AND_TECHNICAL_RESEARCH.md
```

## Related Folders

- **`/reference-apps-copies/`** - Contains actual copies of reference applications (.app bundles) used for analysis. These include:
  - BetterDisplay.app
  - Display Maid.app
  - Flux.app
  - MonitorControlLite.app
  - Umbra.app

- **`/SuperDimmer-Mac-App/`** - The main Mac application source code
