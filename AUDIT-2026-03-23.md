## Product Summary

**SuperDimmer** is a macOS menu bar utility that provides **intelligent, region-specific screen dimming** — the first app to detect and dim only bright content areas (white emails in dark mode, bright webpages, etc.) rather than dimming the entire screen uniformly.

- **Platform**: macOS 13.0+ (Ventura and later)
- **Stack**: Swift 5.9+, SwiftUI + AppKit
- **Current Version**: v1.0.5 (released 2026-02-18)
- **Business Model**: Freemium — Free tier + Pro license at $12 one-time payment
- **Distribution**: Direct download (DMG) from website, not Mac App Store

## Features (Current)

Based on codebase analysis:

- **Bright Region Detection**: BrightnessAnalysisEngine + BrightRegionDetector for real-time screen content analysis
- **Overlay Dimming**: DimOverlayWindow + OverlayManager for targeted region overlays
- **Color Temperature**: ColorTemperatureManager for night shift-style warmth
- **Menu Bar Control**: SwiftUI menu bar interface with dimming slider
- **App Exclusions**: Per-app exclusion list in preferences
- **Auto-Hide**: Smart auto-hide with space awareness and inactivity tracking
- **Launch at Login**: Native launch-at-login support
- **Super Spaces**: Per-desktop-space dimming configurations
- **Accessibility Focus**: Observer for accessibility-aware dimming behavior
- **Auto Updates**: Sparkle-based update mechanism (documented but needs verification)
- **Logging**: AppLogger for diagnostics

## What's Needed for Monetization

### Priority 1: Payment + Licensing Integration (Critical Path to Revenue)

1. **Choose payment provider**: Paddle SDK is already researched and recommended in docs. $12 one-time price is set.
2. **Implement license key validation**: Gate Pro features (auto-hide, Super Spaces, app exclusions, color temperature) behind license check.
3. **Add license activation UI**: In-app purchase flow or redirect to Paddle checkout.
4. **Update website "Get Pro License" button**: Link to actual Paddle checkout page instead of `/#download`.
5. **Add license key entry in Preferences**: Allow users to paste/enter their license key.

### Priority 2: Distribution Improvements

6. **Consider Mac App Store listing**: Broader discovery, built-in payment (30% cut vs Paddle's 5%), but gains visibility.
7. **Notarization**: Ensure DMG is properly notarized for Gatekeeper (may already be done — `Certs/` folder has Developer ID certificate).
8. **Auto-update verification**: Confirm Sparkle auto-update is functional so users get patches.

### Priority 3: Growth

9. **Analytics**: Add basic anonymous telemetry (opt-in) to understand feature usage.
10. **Review/testimonial collection**: Add "Rate SuperDimmer" prompt after N days of use.
11. **Blog SEO**: Continue blog content strategy (already 8+ articles — good).
12. **Product Hunt / Hacker News launch**: One-time visibility boost.

## Recommendation

**SuperDimmer is the most monetization-ready project in the entire portfolio.** It has:
- A unique, defensible product (first-of-kind region-specific dimming)
- Professional website with pricing page
- Active development (79 commits, v1.0.5)
- Clear market positioning and competitive analysis
- A set price ($12 lifetime)

**The single blocker is: no payment integration.** Implementing Paddle SDK licensing is estimated at 2-3 days of focused work and would immediately enable revenue. This should be the highest-priority task for the swarm.
